Descargas

Tal vez quiera descargar los datos que aparecen en la tabla de la pantalla para realizar análisis posteriores. Puede elegir entre dos formatos de archivo.
En cada caso los datos se mostrarán en forma de tabla con los títulos apropiados, tal como aparecen en pantalla. Si la tabla está paginada, todos los datos de todas las páginas se descargarán a un único archivo.

Formato de hoja de cálculo Excel:

Se descargará una hoja de cálculo .xls.

Formato de texto

En este caso, los datos se almacenarán en formato texto. Cada fila de la tabla equivaldrá a una fila de datos separados por tabuladores.
