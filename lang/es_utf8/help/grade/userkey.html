Clave de usuario

La publicación de calificaciones es una forma de exportar las calificaciones vía URL sin tener que autentificarse en Moodle. Una clave de usuario es un conjunto de caracteres generados aleatoriamente que se agrega a la URL para garantizar la seguridad.

Para publicar las calificaciones, seleccione "Crear una nueva clave de usuario". Cuando se envíe el formulario se creará una nueva clave de usuario. Alternativamente, podrá seleccionarse una clave creada con anterioridad.
