Paquetes de contenido IMS

    Este tipo de recurso le permite agregar a su curso paquetes que se ajustan a las especificaciones de paquete de contenido IMS.

    Simplemente suba un paquete comprimido y despliéguelo.

    Tiene disponibles varias opciones para mostrar el contenido en una ventana emergente, con un menú de navegación, botones, etc.
