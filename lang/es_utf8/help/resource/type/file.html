Archivos y páginas web

    Este tipo de recurso permite enlazar cualquier página web u otro archivo de la web pública. También permite enlazar con cualquier página web u otro archivo que usted haya subido al área de archivos de su curso desde su propio ordenador personal.

    Las páginas web normales se muestran tal cual, en tanto que los archivos multimedia se tratan de modo inteligente y pueden incrustarse dentro de una página web. Por ejemplo, los archivos MP3 pueden mostrarse utilizando un reproductor incorporado, así como los archivos de video, animaciones flash y así sucesivamente.

    Se dispone de muchas opciones para mostrar su contenido en ventanas emergentes, ventanas con marcos, etc.

    En concreto, si su recurso es una apliación web u otro tipo de contenido capaz de aceptar parámetros, usted puede elegir enviar información a su recurso tal como el nombre de usuario, su dirección de correo, el curso en que está matriculado, etc.
