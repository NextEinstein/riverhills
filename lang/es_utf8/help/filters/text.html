Filtro de Texto

Este filtro permite filtrar información basada en un formulario de texto libre. El filtro tiene las opciones siguientes:

  • contiene - esta opción admite únicamente información que contiene el texto introducido (si no se ha escrito texto, el filtro queda desactivado)
  • no contiene - esta opción admite únicamente información que no contiene el texto introducido (si no se ha escrito texto, el filtro queda desactivado)
  • es igual a - esta opción admite únicamente información igual al texto introducido (si no se ha escrito texto, el filtro queda desactivado
  • comienza con - esta opción admite únicamente información que comienza con el texto introducido (si no se ha escrito texto, el filtro queda desactivado)
  • termina con - esta opción admite únicamente información que termina con el texto introducido (si no se ha escrito texto, el filtro queda desactivado)
  • está vacío - esta opción admite únicamente información que es igual a una cadena vacía (el texto introducido es pasado por alto)
