Filtro de Perfil

Esta opción le permite filtrar a los usuarios sobre la base de los valores de los campos de perfil. El filtro puede aplicarse a un solo o a todos los campos de perfil. El filtro tiene las opciones siguientes:

  • contiene - esta opción admite únicamente a los usuarios para quienes el campo especificado contiene el texto introducido (si no se ha escrito texto, el filtro queda desactivado)
  • no contiene - esta opción admite únicamente a los usuarios para quienes el campo especificado no contiene el texto introducido (si no se ha escrito texto, el filtro queda desactivado)
  • es igual a - esta opción admite únicamente a los usuarios para quienes el campo especificado es igual al texto introducido (si no se ha escrito texto, el filtro queda desactivado)
  • comienza con - esta opción admite únicamente a los usuarios para quienes el campo especificado comienza con el texto introducido (si no se ha escrito texto, el filtro queda desactivado)
  • termina con - esta opción admite únicamente a los usuarios para quienes el campo especificado termina con el texto introducido (si no se ha escrito texto, el filtro queda desactivado)
  • está vacío - esta opción admite únicamente a los usuarios para quienes el campo especificado es igual a una cadena vacía pero está definida (el texto introducido es pasado por alto)
  • no está definido - esta opción admite únicamente a los usuarios para quienes el campo especificado no está definido (el texto introducido es pasado por alto)
  • está definido - esta opción admite únicamente a los usuarios para quienes el campo especificado está definido (el texto introducido es pasado por alto)
