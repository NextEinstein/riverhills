Subir grupos

Esta opción permite subir grupos a Moodle.

  • Cada línea del archivo contiene un registro
  • Cada registro es una serie de datos separados por comas
  • El primer registro del archivo es especial, y contiene una lista con los nombres de campo. Este registro define el formato del resto del archivo.

    Campos necesarios: estos campos deben incluirse en el primer registro, y deberán ser definidos por cada usuario

    groupname

    Nombres de campo por defecto: son opcionales - si no se incluyen, los valores se toman del idioma y curso actuales

    idnumber, coursename, lang

    Nombres de campo opcionales: todos son completamente opcionales.

    description, picture, hidepicture

  • Las comas deberían codificarse como &#44 - el script automáticamente convertirá dichos códigos en comas.
  • En campos booleanos, use 0 para falso y 1 para verdadero.
  • Para identificar el curso se usará bien idnumber, bien coursename. Idnumber prevalece sobre coursename. Si no se especifica nada, los grupos se agregarán al curso actual.
  • Coursename es el nombre corto del curso.
  • Nota: Si un grupo ya está registrado en la base de datos de Moodle en un curso particular, este script devolverá el nombre de grupo de ese grupo. Los profesores sólo podrán subir grupos a cursos para los que tienen permiso de edición.

He aquí un ejemplo de archivo válido:

groupname,idnumber,lang,description,picture
group1, Phil101, es, ¡Este grupo requiere atención extra!, 0
group2, Math243, , ,
