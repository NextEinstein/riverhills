Modo HTML

Usted dispone de las siguientes posibilidades:

No HTML
No usa marcas HTML. El formateado se realiza mediante PalabrasWiki ('WikiWords'). Los filtros trabajan en este modo.
HTML seguro
Se permiten y se muestran algunas marcas. No hay filtros activos que necesiten marcas en este modo!
Sólo HTML
No se usan PalabrasWiki ('WikiWords'), sólo HTML. Esta opción permite el uso del editor HTML.
